# AI Powered Recipe Generator

## Project Description
AI Recipe Generator is an AI-powered web application developed using Python and Streamlit. It helps users generate unique recipes based on ingredients or cuisine preferences using generative AI.

The application generates step-by-step cooking instructions instantly, making cooking easier and more innovative.

## Features
• Generate recipes using AI  
• User-friendly interface with Streamlit  
• Ingredient based recipe suggestions  
• Instant cooking instructions  
• Fast and responsive design  

## Technologies Used
Python  
Streamlit  
Generative AI API (Gemini/OpenAI)
